## Installation

Install Gosu dependencies (macOS).

```sh
brew install sdl2
```

For Linux dependencies, read [Getting Started on Linux](https://github.com/gosu/gosu/wiki/Getting-Started-on-Linux).

Install gems.

```sh
bundle install
```

Start.

```sh
ruby raycaster.rb
```

Use WASD or arrows to move.
